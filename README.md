# Product-Sales-Analysis-
This is  Flexible data comparison and analytics dashboard built in Python using Streamlit. It automatically understand structure, detect the main comparison entity, and generate analysis, charts, trends, and insights without changing the source code for each new dataset. It has been designed as a generic analytics system.
Expanding this to ML Project will create product optimization for maximum sales. It will analyze previous sales and provide insights for maximum profit. Used scikitlearn for ML and AI based optimized sales price
